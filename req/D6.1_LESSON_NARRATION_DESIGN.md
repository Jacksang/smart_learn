# D6.1 Lesson Narration - Design Document

## Purpose
This document describes Smart Learn's Lesson Narration system - converting learning materials to audio using TTS technology with background music for multi-modal learning experiences.

## Overview
While most students learn primarily through reading, some prefer auditory learning or learn better with background music. This feature:

1. **Converts** learning materials to speech using text-to-speech (TTS)
2. **Supports** multiple voices for variety and personalization
3. **Integrates** background music during narration
4. **Controls** speed for comfortable listening

This creates an accessible, multi-modal learning experience that supports different learning styles.

## Component 1: TTS Engine Interface

### Engine Options

**1. ElevenLabs API (Premium)**
- **Best quality:** Ultra-realistic AI voices
- **Features:** Voice cloning, emotion, stability
- **Speed:** Fast generation (seconds)
- **Cost:** Pay-per-character

**2. Local TTS (Free)**
- **Voice quality:** Good but less natural
- **Features:** Multiple voices, offline capable
- **Speed:** Slower (minutes for long content)
- **Cost:** Free

**3. System TTS (Fallback)**
- **Voice quality:** Basic but reliable
- **Features:** System voices, universal support
- **Speed:** Variable
- **Cost:** Free

### Text Preprocessing

**Smart Text Normalization:**
```javascript
function preprocessText(text) {
  return text
    .replace(/\b\d+\b/g, (match) => formatNumber(match))
    .replace(/\b(19|20)\d{2}\b/g, (match) => formatYear(match))
    .replace(/&/g, ' and ')
    .replace(/%/g, ' percent ')
    .replace(/\$/g, ' dollars ')
    .replace(/\betc\b/g, ' et cetera ')
    .replace(/\bvs\b/gi, ' versus ')
    .replace(/\bno\b/g, ' no ')
    .replace(/\byes\b/g, ' yes ');
}
```

**Punctuation Markers for Better Prosody:**
- `,` → Short pause (0.1s)
- `.` → Medium pause (0.3s)
- `...` → Long pause (0.5s)
- `...` → Section break (1.0s)
- Newlines → Paragraph break (0.5s)

## Component 2: Audio Mixer

### Background Music Integration

**Music Styles:**
1. **Calm:** Soft piano, ambient
2. **Upbeat:** Light pop, acoustic
3. **Classical:** Bach, Mozart, Beethoven
4. **Jazz:** Smooth jazz, lounge
5. **Nature:** Birds, rain, waves
6. **Silence:** No background music

**Volume Mixing:**
- Music volume: 0-100% (default: 30%)
- Speech volume: 100% (default)
- Ducking ratio: 70% (music lowers to 30% during speech)

**Ducking Algorithm:**
```javascript
function applyDucking(speechLevels, musicLevels, duckRatio) {
  return musicLevels.map((level, i) => {
    if (speechLevels[i] > threshold) {
      return level * duckRatio; // Lower music during speech
    }
    return level; // Normal music volume
  });
}
```

**Audio Mixing Pipeline:**
```javascript
class AudioMixer {
  async mixNarration(narrationAudio, backgroundMusic, options) {
    // Normalize volumes
    const normalizedSpeech = normalizeVolume(narrationAudio, 1.0);
    const normalizedMusic = normalizeVolume(backgroundMusic, 0.3);
    
    // Apply ducking
    const duckedMusic = applyDucking(
      normalizedSpeech.levels,
      normalizedMusic.levels,
      0.3  // Music drops to 30% during speech
    );
    
    // Mix audio tracks
    const mixedAudio = await ffmpegMix(
      normalizedSpeech,
      duckedMusic
    );
    
    return mixedAudio;
  }
}
```

## Component 3: Narration Service

### Service Functions

**1. Generate Narration**
- Convert lesson materials to speech
- Add background music layer
- Mix audio tracks
- Return downloadable audio file

**2. Get Narration**
- Retrieve generated narration by ID
- Include metadata and playback info
- Provide streaming URL

**3. List Narrations**
- Get all narrations for a project/lesson
- Show status (generating, complete, failed)
- Include duration and file size

**4. Stop Narration**
- Cancel ongoing generation
- Clean up temporary files
- Free resources

**5. Update Narration Settings**
- Change voice, speed, music style
- Regenerate with new settings
- Cache previous versions

### Service Class
```javascript
class NarrationService {
  constructor() {
    this.ttsEngine = new TTSEngine();
    this.audioMixer = new AudioMixer();
    this.narrations = new Map();
  }

  async generateNarration(projectId, lessonId, materials, options) {
    // Preprocess text
    const processedText = this._preprocessText(materials);
    
    // Generate speech
    const speechAudio = await this._generateSpeech(processedText, options.voice);
    
    // Add background music
    const backgroundMusic = this._getBackgroundMusic(options.music);
    
    // Mix audio
    const mixedAudio = await this.audioMixer.mixNarration(
      speechAudio,
      backgroundMusic,
      options.mixing
    );
    
    // Save narration
    const narration = this._saveNarration({
      projectId,
      lessonId,
      audioFile: mixedAudio,
      options,
      status: 'complete'
    });
    
    return narration;
  }
}
```

### Performance Optimization

**Caching:**
- Hash-based audio caching

**Streaming:**
- Chunked generation for large files

**Async Processing:**

**Performance Targets:**

## Component 4: Playback Integration

### Playback Features

**Speed Control:**
- Playback speed: 0.5x to 2.0x
- Default speed: 1.0x
- Smooth speed transitions

**Chapter Navigation:**
- Jump to specific sections
- Chapter markers in audio
- Skip forward/backward by time

**Progress Tracking:**
- Last position bookmarking
- Auto-resume from interruption
- Progress bar with timestamps

### Playback Interface
```javascript
class LessonPlayer {
  constructor(narrationId) {
    this.narration = this._loadNarration(narrationId);
    this.currentTime = 0;
    this.playbackRate = 1.0;
    this.isPlaying = false;
    this.bookmarks = [];
  }

  play() {
    this.audio.playbackRate = this.playbackRate;
    this.audio.play();
    this.isPlaying = true;
    this._startProgressTracking();
  }

  pause() {
    this.audio.pause();
    this.isPlaying = false;
    this._saveBookmark();
  }

  seek(seconds) {
    this.audio.currentTime = seconds;
    this.currentTime = seconds;
  }

  setSpeed(rate) {
    this.playbackRate = Math.max(0.5, Math.min(2.0, rate));
    this.audio.playbackRate = this.playbackRate;
  }
}
```
