# D6.2 Voice Interaction - Design Document

## Purpose
This document describes Smart Learn's Voice Interaction system - enabling students to speak naturally during lessons using speech recognition (Whisper CLI) for a conversational learning experience.

## Overview
While D6.1 (Lesson Narration) converts text to speech for **listening**, D6.2 converts speech to text for **speaking**. This feature:

1. **Captures** voice input from students via microphone or audio file
2. **Transcribes** speech to text using Whisper CLI
3. **Recognizes** voice commands for lesson control
4. **Interprets** natural language questions and comments

This creates a hands-free, conversational learning experience that feels more natural and accessible.

## Component 1: Whisper CLI Integration

### Integration Approach

**Whisper CLI:**
- Install via pip: `pip install openai-whisper`
- Command: `whisper audio.mp3 --model medium`
- Output: JSON with transcript, timestamps, confidence
- Models: tiny, base, small, medium, large

### Engine Interface
```javascript
class WhisperEngine {
  // Initialize CLI path
  async initialize(cliPath)
  
  // Transcribe audio file
  async transcribe(audioPath, options)
  
  // Get available models
  async getModels()
  
  // Cancel transcription
  async cancel(jobId)
}
```

### CLI Command Structure
```bash
whisper {audio_path} \
  --model {model_size} \
  --output_format json \
  --output_dir {output_dir} \
  --temperature 0 \
  --without_timestamps
```

- MP3 (most compatible)
- WAV (high quality)
- M4A (iPhone format)
- FLAC (lossless)
- OGG (efficient)

### Error Handling & Retry Logic
- File not found
- Unsupported audio format
- Whisper not installed
- Out of disk space
- Invalid model name

**Retry with backoff:**
```javascript
async function transcribeWithRetry(audioPath, options, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      return await whisperEngine.transcribe(audioPath, options);
    } catch (error) {
      if (i === maxRetries - 1) throw error;
      await wait(1000 * (i + 1)); // Exponential backoff
    }
  }
}
```

### Command Parsing Algorithm

**Intent Classification:**
```javascript
function parseVoiceCommand(text) {
  const lowerText = text.toLowerCase().trim();
  
  // Playback commands
  if (lowerText.includes('play') || lowerText.includes('start')) {
    return { intent: 'playback', action: 'play', params: {} };
  }
  if (lowerText.includes('pause')) {
    return { intent: 'playback', action: 'pause', params: {} };
  }
  
  // Navigation commands
  if (lowerText.includes('next') || lowerText.includes('skip')) {
    return { intent: 'navigation', action: 'next', params: {} };
  }
  if (lowerText.includes('back') || lowerText.includes('previous')) {
    return { intent: 'navigation', action: 'previous', params: {} };
  }
  
  // Speed commands
  if (lowerText.includes('faster') || lowerText.includes('speed up')) {
    return { intent: 'speed', action: 'increase', params: { amount: 0.25 } };
  }
  if (lowerText.includes('slower') || lowerText.includes('slow down')) {
    return { intent: 'speed', action: 'decrease', params: { amount: 0.25 } };
  }
  
  // Default: Free-form question
  return { intent: 'question', text: lowerText };
}
```

### Question Detection

**Intent Detection:**
```javascript
function isQuestion(text) {
  const questionPatterns = [
    /^what\b/i, /^how\b/i, /^why\b/i,
    /^when\b/i, /^where\b/i, /^who\b/i,
    /\?$/, /can you/i, /could you/i,
    /is it/i, /is this/i
  ];
  
  return questionPatterns.some(pattern => pattern.test(text));
}

function classifyQuestion(text) {
  if (/what\b/i.test(text)) return 'definition';
  if (/how\b/i.test(text)) return 'process';
  if (/why\b/i.test(text)) return 'reasoning';
  if (/when\b/i.test(text)) return 'temporal';
  if (/where\b/i.test(text)) return 'location';
  if (/who\b/i.test(text)) return 'entity';
  return 'general';
}
```

## Component 3: Voice Quiz Service

### Service Functions

**1. Record Answer**
- Capture voice input
- Transcribe via Whisper
- Return transcribed text

**2. Evaluate Answer**
- Compare transcribed answer to expected
- Calculate similarity score
- Determine correctness

**3. Provide Feedback**
- Audio feedback response
- Text feedback
- Encouraging messages

**4. Track Progress**
- Voice interaction metrics
- Answer history
- Performance analytics

### Answer Evaluation

**Similarity Metrics:**
- **Exact Match:** Perfect string equality (100%)
- **Jaccard Similarity:** Shared words (0-100%)
- **Semantic Similarity:** Vector-based meaning match (0-100%)
- **Edit Distance:** Character-level differences (0-100%)

**Scoring Algorithm:**
```javascript
function calculateSimilarity(answer1, answer2) {
  const words1 = answer1.toLowerCase().split(/\s+/);
  const words2 = answer2.toLowerCase().split(/\s+/);
  
  const intersection = words1.filter(word => words2.includes(word));
  const union = [...new Set([...words1, ...words2])];
  
  const jaccard = intersection.length / union.length;
  return jaccard * 100; // Normalize to 0-100 scale
}
```

CREATE TABLE voice_recordings (
  id UUID PRIMARY KEY,
  session_id UUID,
  question_id UUID,
  audio_path TEXT,
  transcript TEXT,
  language VARCHAR(10),
  confidence DECIMAL(3,2),

CREATE TABLE voice_interactions (
  id UUID PRIMARY KEY,
  session_id UUID,
  recording_id UUID REFERENCES voice_recordings(id),
  intent_type VARCHAR(50),
  command_action VARCHAR(50),
  answer_correct BOOLEAN,

  language VARCHAR(10) DEFAULT 'en',
  whisper_model VARCHAR(20) DEFAULT 'small',
  auto_transcribe BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW()
);
