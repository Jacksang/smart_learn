# D3.3 Transcript/Audio/Video Support - Function Breakdown

## Purpose
Implement comprehensive transcript, audio, and video file parsing support for Smart Learn.

## Business Logic
- Parse subtitle files (.srt, .vtt) with timestamp extraction
- Integrate Whisper CLI for audio transcription  
- Extract video frames and metadata for video files
- Store parsed content with timestamps and metadata in PostgreSQL

## Implementation Tasks

### D3.3.1 Subtitle file parsing ✅ COMPLETE
- [x] Parse SRT (SubRip Subtitle) files
  - Extract timestamp blocks and text content
  - Parse format: `HH:MM:SS,mmm --> HH:MM:SS,mmm`
  - Return structured data with text and timestamps
- [x] Parse VTT (WebVTT) files
  - Handle WEBVTT header and timestamp format: `HH:MM:SS.mmm --> HH:MM:SS.mmm`
  - Extract text and timing information
- [x] Parse plain text files (.txt)
  - Simple text extraction without timestamps
