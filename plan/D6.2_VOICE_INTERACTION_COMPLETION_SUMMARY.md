## Overview

D6.2 Voice Interaction successfully implements a conversational learning experience that:

1. **Captures** voice input from students via microphone or audio file
2. **Transcribes** speech to text using Whisper CLI
3. **Recognizes** voice commands for lesson control
4. **Interprets** natural language questions and comments

This creates a hands-free, interactive learning experience with natural command support.

**Similarity Calculation:**
```javascript
function calculateSimilarity(answer1, answer2) {
  const words1 = answer1.toLowerCase().split(/\s+/);
  const words2 = answer2.toLowerCase().split(/\s+/);
  
  const intersection = words1.filter(word => words2.includes(word));
  const union = new Set([...words1, ...words2]);
  
  const jaccard = intersection.length / union.length;
  return jaccard * 100; // 0-100 scale
}
```
