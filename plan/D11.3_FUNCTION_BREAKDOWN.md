- [x] **A1** — `backend/migrations/20260430_add_session_management.sql` ✅

### Phase B: Repository Layer
- [x] **B1** — `backend/src/sessions/repository.js` — 11 new functions ✅

  - `getProgressSnapshots(sessionId, limit)` — SELECT recent snapshots
  - `createSessionSummary(sessionId, summaryData)` — INSERT into session_summaries
  - `getSessionSummary(sessionId)` — SELECT latest summary
  - `recordModeSwitch(sessionId, fromMode, toMode, reason)` — INSERT into session_mode_history
  - `getModeHistory(sessionId)` — SELECT mode history ordered by timestamp
