## Technical Implementation

### Parser Interface
```javascript
async function parse(filePath) {
  return {
    text: string,                    // Extracted transcript text
    timestamps: [],                  // Word-level timestamps
    metadata: {                      // File metadata
      fileName: string,
      fileType: string,              // e.g., '.mp3'
      fileSize: number,
      duration: string,              // Audio duration
      processedAt: string            // ISO timestamp
    }
  };
}
```

### Whisper CLI Integration
- **Command:** `whisper <file> --output-format txt --model base --word-timestamps`
- **Model:** Base model for balance of speed/accuracy
- **Output:** Word-level timestamps with text content
