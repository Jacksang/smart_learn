## Technical Specifications

### Parser Interface

All parsers follow the same interface:

```javascript
async function parse(filePath) {
  return {
    // For text/audio/video with timecodes
    text: string,           // Extracted content
    timestamps: [],         // Optional timing information
    
    // Frames for video
    frames: [],             // Optional frame array
    
    // Standard metadata
    metadata: {
      fileName: string,
      fileType: string,
      fileSize: number,
      duration: string,     // For audio/video
      width: number,        // For video
      height: number,       // For video
      codec: string,        // For video
      frameRate: string,    // For video
      processedAt: string
    }
  };
}
```

### Service Routing

```javascript
async function extractFileContent(filePath, mimeType) {
  const extension = path.extname(filePath).toLowerCase();
  
  // Route to appropriate parser based on file type
  if (extension === '.csv') return csvParse(filePath);
  if (extension in ['.xls', '.xlsx']) return excelParse(filePath);
  if (extension in ['.srt', '.vtt', '.txt']) return transcriptParse(filePath);
  if (extension in ['.mp3', '.wav', '.m4a', '.flac', '.ogg']) return audioParse(filePath);
  if (extension in ['.mp4', '.avi', '.mov', '.mkv']) return videoParse(filePath);
  
  throw new Error(`Unsupported file type: ${extension}`);
}
```

## Known Limitations

1. **External Dependencies Required:**
   - Whisper CLI for audio transcription
   - FFmpeg for video processing
   - Not all environments may have these installed

2. **Processing Time:**
   - Audio transcription depends on duration
   - Video processing can be slow for long videos
   - Consider async background jobs for large files

3. **Frame Extraction:**
   - Adaptive extraction may miss key moments in fast-paced videos
   - Fixed frame intervals, not scene-aware
