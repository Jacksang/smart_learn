# D4.2 Adaptive Encouragement - Function Breakdown

## Purpose
Enhance the Motivation Engine with adaptive, situation-aware encouragement that detects struggle signals, provides confidence-support messaging, and recommends appropriate next steps.

## Business Logic
- Detect when students are experiencing struggle (frustration, repeated failures, long pauses)
- Provide confidence-boosting messages to support students in difficult moments
- Recommend appropriate next steps based on the detected struggle pattern
- Learn from student responses to improve future recommendations
