## Technical Implementation

### Parser Interface
```javascript
async function parse(filePath) {
  return {
    frames: [],                    // Array of extracted frames
    metadata: {                    // Video metadata
      fileName: string,
      fileType: string,            // e.g., '.mp4'
      fileSize: number,
      duration: string,            // Video duration in seconds
      width: number,               // Video width in pixels
      height: number,              // Video height in pixels
      codec: string,               // Video codec
      frameRate: string,           // Frame rate (e.g., '29.97')
      processedAt: string          // ISO timestamp
    }
  };
}
```

### FFmpeg Integration
- **Command:** `ffmpeg -i <video> -vf fps=<fps> -q:v 2 <output_pattern>`
- **Frame Extraction Strategy:**
  - Videos < 10s: 1 frame every 2 seconds
  - Videos 10-30s: 1 frame every 5 seconds
  - Videos > 30s: 1 frame every 10 seconds
