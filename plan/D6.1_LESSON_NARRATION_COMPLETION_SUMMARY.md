**Engine Interface:**
```javascript
class TTSEngine {
  async initialize(config)     // Initialize with settings
  async generate(text, voiceId, options)  // Generate speech
  async getVoices()            // List available voices
  async cancel(jobId)          // Cancel generation
  async getStatus(jobId)       // Check status
}
```

**Smart Text Preprocessing:**
- Number formatting (1999 → 1999, 75% → 75 percent)
- Currency formatting ($100 → 100 dollars)
- Abbreviation expansion (etc → et cetera, vs → versus)
- Whitespace normalization
- Pause marker insertion (paragraph breaks, commas, periods)

**Voice Configuration:**
```json

  "stability": 0.5,
  "similarity_boost": 0.75,
  "style": 0.0,
  "use_speaker_boost": true

```

**Audio Post-Processing:**
- Volume normalization (0-100%)
- Fade-in/fade-out transitions (default: 0.1s each)
- Peak normalization to prevent clipping
- Format conversion support (MP3, WAV, OGG, FLAC)

### 2. Audio Mixer ✅

**File:** `backend/src/review/audio-mixer.js` (6,231 bytes)

**Core Features:**
- **Volume Normalization:** Target-level scaling with peak detection
- **Intelligent Ducking:** Music lowers during speech (30% during speech, 100% during pauses)
- **Audio Mixing:** Multi-track audio combination
- **Fade Processing:** In/out transitions for smooth playback

**Ducking Algorithm:**
```javascript
function applyDucking(musicData, speechData, duckRatio) {
  return musicData.map((sample, i) => {
    const speechLevel = Math.abs(speechData[i]);
    if (speechLevel > threshold) {
      return sample * duckRatio; // Lower music during speech
    }
    return sample; // Normal volume during pauses
  });
}
```

**Volume Levels:**
- **Speech:** 0dB (100%)
- **Music:** -10dB (30%)
- **Ducking:** Music drops to -20dB (10%) during speech

**Available Music Styles:**
1. **Calm:** Soft piano and ambient (pink noise blend)
2. **Upbeat:** Light and energetic (higher intensity)
3. **Classical:** Bach, Mozart, Beethoven (soft blend)
4. **Jazz:** Smooth jazz and lounge (gentle rhythm)
5. **Nature:** Birds, rain, ocean waves (organic sounds)
6. **Silence:** No background music

**Audio Generation:**
- `generateSilentMusic(duration)`: Pure silence
- `generatePinkNoise(duration, volume)`: Soft background noise
- Pink noise formula: `output = (lastOut + 0.02 * whiteNoise) / 1.02`

**Performance:**
- Normalization: < 100ms
- Ducking: < 200ms
- Mixing: < 300ms
- Total: < 5 seconds

**Testing:** Volume normalization accuracy, ducking calculation, fade application, mixing logic, music style generation
