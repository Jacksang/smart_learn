# D7.2 Dashboard and Visualization - Function Breakdown

## Purpose
This document defines the implementation tasks for Smart Learn's Progress Dashboard and Visualization feature (D7.2) - frontend interfaces for displaying learning progress and analytics.

---

## Overview

While D7.1 provides the backend analytics engine, D7.2 adds the **user-facing** component - interactive dashboards and visualizations that make progress data accessible, actionable, and engaging for students. This feature:

1. **Displays** real-time progress metrics with charts and graphs
2. **Visualizes** mastery levels with color-coded indicators
3. **Highlights** weak areas needing attention
4. **Presents** personalized recommendations in a clear format
5. **Enables** interactive filtering and time-range selection

This creates an intuitive user interface that helps students understand their learning journey and make informed decisions about their study focus.

---

```jsx
function MetricCard({ title, value, trend, trendDirection }) {
  const trendColor = trendDirection === 'up' ? 'green' : 
                     trendDirection === 'down' ? 'red' : 'gray';

  return (
    <div className="progress-card">
      <h3>{title}</h3>
      <div className={`metric-value ${trendDirection}`}>
        {value}
      </div>
      <div className="metric-trend trend-{trendColor}">
        {trend}
      </div>
    </div>
  );
}
```

### Display Logic
- **Accuracy:** Color-coded (green ≥80%, yellow ≥60%, red <60%)
- **Response Time:** Direction indicator (↓ = faster, ↑ = slower)
- **Streak:** Fire emoji for active streak, celebration for milestones
- **Session Duration:** Format as minutes/hours

## API Integration

### Data Fetching
```jsx
async function fetchProgressData(sessionId) {
  try {
    const [
      sessionProgress,
      learningProgress,
      weakAreas,
      recommendations
    ] = await Promise.all([
      fetch(`/api/projects/123/review/progress/session/${sessionId}`),
      fetch(`/api/projects/123/review/progress/learning`),
      fetch(`/api/projects/123/review/progress/weak-areas`),
      fetch(`/api/projects/123/review/progress/recommendations`, {
        method: 'POST',
        body: JSON.stringify({ focusAreas: ['weak', 'review'] })
      })
    ]);

    return {
      session: await sessionProgress.json(),
      learning: await learningProgress.json(),
      weak: await weakAreas.json(),
      recs: await recommendations.json()
    };
  } catch (error) {
    console.error('Failed to fetch progress:', error);
    return null;
  }
}
```

### Loading States
```jsx
function ProgressDashboard({ sessionId }) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const loadData = async () => {
      try {
        const result = await fetchProgressData(sessionId);
        setData(result);
      } catch (err) {
        setError('Failed to load progress data');
      } finally {
        setLoading(false);
      }
    };

    loadData();
  }, [sessionId]);

  if (loading) return <ProgressLoading />;
  if (error) return <ErrorDisplay message={error} />;
  if (!data) return <NoDataDisplay />;

  return (
    <div className="progress-dashboard">
      <ProgressOverview data={data.learning} />
      <MasteryBreakdown levels={data.learning.masteryLevels} />
      <ProgressTrends data={data.learning.dailyProgress} />
      <WeakAreas areas={data.weak.areas} />
      <Recommendations recs={data.recs.recommendations} />
    </div>
  );
}
```

### Loading States
- Skeleton screens for all cards
- Smooth transitions
- Auto-refresh button
- "Last updated" timestamp

### Error States
- Graceful error messages
- Retry button
- Data availability indicator
- Fallback to cached data if available

### Data Export
```jsx
<button className="export-btn" onClick={handleExport}>
  📥 Export Progress Report
</button>

async function handleExport() {
  const data = {
    overview: progressData.overview,
    masteryLevels: progressData.masteryLevels,
    trends: progressData.dailyProgress,
    exportedAt: new Date().toISOString()
  };

  const blob = new Blob([JSON.stringify(data, null, 2)], {
    type: 'application/json'
  });
  
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = `progress-report-${sessionId}.json`;
  a.click();
}
```

### Real-Time Updates
```jsx
useEffect(() => {

  return () => clearInterval(interval);

// Also refresh on user interaction
const handleAnswerSubmit = async (answerData) => {
  await submitAnswer(answerData);
```
